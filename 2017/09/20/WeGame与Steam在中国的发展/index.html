WeGame与Steam在中国的发展

发表于 2017-09-20

WeGame的成立，可以说是正式点燃了国内PC游戏市场的热情。我作为一个PC游戏老司机听到这个消息后非常激动，这是国内PC市场回暖的一个重要标志。Steam由于目前并未正式入华，其运营模式目前处于灰色地带，但不可否认，Steam在中国市场取得了令人羡慕的成功。目前Steam的中国用户数占比已经跃居第三位。



图一：Steam 用户数占比 (数据采集自steamspy)


图二：Steam 用户过去两周游戏市场(数据采集自steamspy)

从图上可以看出，steam在中国市场已经取得了巨大的成功，PC游戏这块蛋糕也是越做越大，一定上已经挤占了国内传统网络游戏的市场份额。截止发出文章前，Steam在中国区的活跃用户已经达到了2194 5054人。
从Steam的人员构成来看，最早进入Steam的以硬核玩家为主，其中有大量的ACG爱好者。这些人的消费能力很强。我曾经看过国区很多Steam等级过100的大佬，有很大一部分拥有大量游戏和小姐姐壁纸，这些东西价格不菲。下面我首先说一下Steam的发展战略。

Steam发展战略

  1. 国区低价区的定位。 Steam国区定位为低价区，这是Steam能在国内市场上蓬勃发展的重要一点。Steam上游戏的价格按美区来讲，远远超过了国人的平均消费水平。除了硬核玩家外，合理的游戏价格才能扩展用户。这一点从最近大火的大逃杀就可以看出。大逃杀美区售价为$29.99，合人民币197元，国区售价仅为98元人民币，全球最低。这一定价策略，极大的拓展了用户数，加上游戏本身不俗的素质与直播的推波助澜之下，这款游戏已经成为了目前国内最火的游戏。甚至连我的一部分LOL用户都被“吃鸡”所吸引。低价战略对于WeGame的发展应该很多启示，我认为重要的一点就是要坚持游戏本体价格合理，游戏内额外内购增值的方法。
  2. 通过平台入手，逐渐整合市场。很多人进入Steam，都是被最初几款大作引入的。像2011年的老滚5, 2012年的Dota2，以及后来搭上直播东风的GTA5，都让人首先了解了这个平台。Steam本身再通过两条线吸引玩家，一条是大量的新作，一条是Steam本身提供的交易市场。市场通过便捷而且公平的方式为玩家提供了优秀的交易体验，新作和覆盖面相当广的大作（除EA都有），让玩家产生了在平台购买游戏稳定且有保障的感觉。
  3. 上乘的购买体验。G胖曾经说过，如果正版比盗版体验更好，那么玩家自然会选择正版游戏，事实证明了这一点。过去很多游戏或者平台，有着像UPlay一样糟糕的体验，让玩家成为了正版软件的受害者。而Steam从游戏备份，服务器的稳定性和遍布全球的CDN等各个方面，为玩家提供了傻瓜式的体验。过去我安装盗版游戏的时候，经常会为下载了很久的游戏能否正确安装而犯愁，时常下载安装后缺少运行库，而找运行库的过程十分煎熬，往往找到了也会出现版本不匹配，无法正常游戏的问题。Steam让我正式放弃了盗版游戏，不管什么作品，盗版游戏我都已经不会再去下载了，投入了一年两次的剁手大军中。
    此外，Steam接入了多种多样的支付方式，特别对于中国玩家接入了微信支付，支付宝和银联支付，顺滑的支付体验极大的增加了玩家的购买欲望。
  4. 便捷的玩家间组队。通过Steam提供的SDK，玩家在游戏中的组队方式得到了很大的优化。对于小白玩家来说，每一个新的游戏如何组队都是一次痛苦的学习过程。Steam内置的组队机制极大的便捷了游戏邀请。让游戏内的社交关系得以稳固，增加了用户黏性。
  5. 定期的特惠。通过定期举行打折的方式，吸引新玩家同时促进玩家回流。特惠期间平台活动（Trading Card）让玩家对平台产生了亲切感，进一步促进了玩家消费。从我本人来说，有了卡片收集作为动力，我今年夏季特惠的剁手值也冲到了新高。
  6. 对于Mod良好的支持。创意工坊是我初入Steam时最为惊叹的一个功能。Mod下载从来没有像在Steam上这么方便过。Mod提升了游戏的活力，大量的UGC也为提高了平台本身的活力。

对于WeGame的启示

  1. 做到合理的游戏定价。这一点在Tencent强大的支持下很有优势，饥荒在WeGame上定价15元，而我在Steam上是以48元购买这款游戏的。有竞争力的价格可以吸引玩家去WeGame。
  2. 腾讯本身强大的平台能力。依托于QQ与微信对国内社交网络的支配地位，WeGame在平台上有先天的优势。此外TGP的运营经验对WeGame来说也有很大帮助。
  3. 优化支付体验。腾讯自有的微信支付需要咋WeGame上做进一步优化。我本人认为可以通过授信额度来提供交易。绑定微信账号的WeGame账号，可以让用户选择开启一定量（300元左右）的授信额度。玩家可以一键付款，无需在电脑端输入密码或者扫码。在一个星期或游戏时间2小时内可以退款，授信额度可以在下一个月通过微信还款。如此畅快的支付方式势必增加玩家的消费。
  4. 游戏组队优化。腾讯可以提供接口，直接将WeGame内组队邀请发送到QQ或者微信，对方玩家若果没有游戏，点开链接会自动下载游戏并组队（或提示购买），有游戏的玩家可以直接组队。
  5. 对Mod有限度的支持。国内对于UGC最重要的一点就是要考虑审查的影响，Mod的审查是一个费时费力的工程。对于Mod开发者，可以采取授权制的方法。只有通过实名认证的玩家才可以上传Mod，并且玩家一旦上传违规Mod，第一次采取警告，第二次封号的方法，可以有效维护平台安全。
  6. 大作的引入。WeGame 平台在发展的过程中，一个绕不过的坎就是大作。如何引入国际大作，如何促进国内大作的发展是重要的问题。引入国际大作还是要依赖于腾讯调动国际资源的能力，国内的大作如果能够良性发展，对于整个游戏生态都会是一个促进。
